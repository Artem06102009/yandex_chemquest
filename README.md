# yandex_chemquest
Описание:
«ChemQuest» – это веб-сайт для тех, кто хочет подтвердить свои скилы в рамках школьного курса химии. Он направлен на привлечение интереса к данному предмету.

Проект содержит разнообразные задания: от составления химических формул до решения задач на основе уравнений реакций. Пользователям доступно семь тематических режимов викторины, включая валентность, степень окисления, типы химической связи, атомную массу, а также определение окислителей и восстановителей.

Игровой процесс построен на двух режимах: «На время» (60 секунд на максимальное количество ответов) и «На количество» (20 вопросов без ограничения времени). Каждый сыгранный результат сохраняется в личном профиле, а лучшие достижения отображаются в общем рейтинге.

Дополнительно проект включает интерактивную таблицу Д. И. Менделеева.

ТЕХНИЧЕСКОЕ ЗАДАНИЕ:

1. Игра
Настройки:
- Режим: На время(60 сек) / На количество(по 20 вопросов)
- Темы: формулы, валентность, степень окисления, типы связи, атомная масса, ОВР
Механика:
- Вопрос + 4 варианта ответа
- После ответа: правильный — зелёный, неправильный — красный
- Переход через 1 секунду
- Сохранение результата в БД (только для авторизованных)

2. Рейтинг
- Топ-10 результатов
- Фильтрация по режиму и теме
- Колонки: место, имя, счёт, игр сыграно

3 Таблица Менделеева
- Интерактивная таблица с символами и названиями элементов, при наведении появляется доп. информация

4. База данных
- User: id, email, password_hash, created_at, games_played
- GameResult: id, user_id, score, mode, theme, played_at

5. Регистрация\Вход
- Регистрация новых пользователей
- Только новые почты
- Пароль от 6 символов
- Повтор пароля
- После регистрации отображаение иконки аккаунта
