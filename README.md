# yandex_chemquest
Описание:
«ChemQuest» – это веб-сайт для тех, кто хочет подтвердить свои скилы в рамках школьного курса химии. Он направлен на привлечение интереса к данному предмету.

Проект содержит разнообразные задания: от составления химических формул до решения задач на основе уравнений реакций. Пользователям доступно семь тематических режимов викторины, включая валентность, степень окисления, типы химической связи, атомную массу, а также определение окислителей и восстановителей.

Игровой процесс построен на двух режимах: «На время» (60 секунд на максимальное количество ответов) и «На количество» (20 вопросов без ограничения времени). Каждый сыгранный результат сохраняется в личном профиле, а лучшие достижения отображаются в общем рейтинге.

Дополнительно проект включает интерактивную таблицу Д. И. Менделеева.
